# Novo 🌐

A console-based Social Media Platform built in C++ as an OOP course project.
FAST-NUCES Lahore | Spring 2026

---

## Features

- User registration, login, logout and profile management
- Create, edit, delete and save posts
- Comment on posts — add, edit and delete
- Feed — view, like and share posts from people you follow
- Search users by username and view their profile
- Private messaging and group chat
- Admin panel — manage reports, delete posts, comments and accounts
- Notifications for likes and comments on your posts
- Password strength checker
- Session time tracker

---

## OOP Concepts

- Composition — Post, Comment and Notification are owned by their parent objects
- Aggregation — Feed, Admin and SearchEngine hold pointers only
- Inheritance — Admin extends User
- Encapsulation — all data members are private
- Dynamic Memory — Rule of Three applied across every class
